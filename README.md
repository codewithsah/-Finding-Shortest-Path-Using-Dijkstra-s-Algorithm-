# Dijkstra's Shortest Path Visualizer

**Interactive Tkinter GUI** that demonstrates Dijkstra’s shortest-path algorithm on a vibrant graph.  
Designed for learning and small demos — select start and end nodes, press *Find Shortest Path*, and watch the path animate.

---

## Features
- Visual, animated highlighting of the computed shortest path.
- Colorful node palette and edge weights displayed on the canvas.
- Start and end nodes selectable from a dropdown.
- Reset button to clear results and restore colors.
- Small, self-contained codebase — easy to extend or adapt.
